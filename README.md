# Personal Website

Krotki projekt portfolio prezentujacy:
- informacje o mnie,
- technologie, z ktorymi pracuje,
- doswiadczenie,
- wybrane projekty,
- szybki kontakt i pobieranie CV.

## Uzyte technologie

- React 19
- TypeScript
- Vite
- Tailwind CSS
- Zustand
- Framer Motion / Motion
- Lucide React + React Simple Icons

## Uruchomienie lokalne

```bash
npm install
npm run dev
```

## Build produkcyjny

```bash
npm run build
npm run preview
```
