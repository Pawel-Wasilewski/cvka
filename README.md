# Personal Website

Krótki projekt portfolio prezentujący:
- informacje o mnie,
- technologie, z którymi pracuje,
- doświadczenie,
- wybrane projekty,
- szybki kontakt i pobieranie CV.

## Uzżte technologie

- React 19
- TypeScript
- Vite
- Tailwind CSS
- Zustand
- Framer Motion / Motion
- Lucide React + React Simple Icons

## Uruchomienie lokalne

```bash
npm install
npm run dev
```

## Build produkcyjny

```bash
npm run build
npm run preview
```
